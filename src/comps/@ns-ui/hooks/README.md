# NsUI Hooks

[![npm](https://img.shields.io/npm/dm/@ns-ui/hooks)](https://www.npmjs.com/package/@ns-ui/hooks)

[View documentation](https://NsUI.dev/)

## Installation

```bash
# With yarn
yarn add @ns-ui/hooks

# With npm
npm install @ns-ui/hooks
```
